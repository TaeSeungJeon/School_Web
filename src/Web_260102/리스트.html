HTML, Request, Response에 대해 알아보자

THML이란 무엇인가?

HTML
Hyper Text Markup Language의 약어이며, 웹페이지의 '뼈대'를 만드는 가장 기초적인 마크업 언어이다.

Request란 무엇인가? (요청)

Request
브라우저(사용자)가 서버에 "나 이거 필요해!"라고 보내는 메세지이다.

Response란 무엇인가? (응답)

Response
서버가 브라우저의 요청을 받고 "여기있어!" 혹은 "그건 없어(404에러)"라고 답하는 메세지이다.
